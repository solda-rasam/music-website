A complete, feature-rich Music Store website containing both a frontend interface for browsing/playing music and a backend system for managing the application.

---

## ⚠️ Important Note (Audio & Image Files)
Due to GitHub's file size limits and to keep this repository lightweight, the **`Audio`** and **`Images`** directories in this main branch do not contain the actual heavy media files. 

**To run the website locally with all songs and pictures:**
1. Go to the **[Releases](https://github.com/solda-rasam/music-website/releases)** section on the right side of this page.
2. Find the latest release (**v1.0.0**).
3. Download the **`musicshop.zip`** asset (approx. 1.56 GB).
4. Extract and place the audio and image files into their respective folders (`Audio/` and `Images/`) inside your local project directory.

---

## 📁 Directory Structure

```text
├── backend/             # Backend server codes and configurations
└── musicshop/           # Frontend source files
    ├── CSS/             # Stylesheets (CSS)
    ├── Fonts/           # Custom website fonts
    ├── HTML/            # HTML pages and layouts
    ├── Audio/           # Audio files folder (Download from Releases)
    └── Images/          # Website images and assets (Download from Releases)

```
## 🚀 Technologies Used
 * **Frontend:** HTML5, CSS3, JavaScript
 * **Backend:**  Python
 * **Version Control:** Git & GitHub
## 💻 How to Run
 1. **Clone or download** this repository to your local machine.
 2. Download the media files from the **Releases** section and place them in the correct folders as explained in the *Important Note* above.
 3. Open the main HTML files inside the musicshop/HTML/ folder in your browser to view the frontend.
 4. Navigate to the backend/ folder and run your local server instructions to start the backend.
